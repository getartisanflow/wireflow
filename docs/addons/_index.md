# Addons

Optional addons that extend the base WireFlow surface with additional capabilities.

| Addon | Purpose |
|-------|---------|
| [Schema](schema.md) | Database-schema designer — Blade components, server trait, validators. |
| [Workflow](workflow.md) | Workflow runs — wait/condition node templates, server-driven runs, run-state sync. |
| [Whiteboard](whiteboard.md) | Freehand drawing primitives layered on the canvas. |
| [Layouts](layouts.md) | Auto-layout integrations (dagre, ELK, force-directed, hierarchy). |
| [Collaboration](collab.md) | Real-time multi-user editing via Yjs CRDTs. |
